How refresh and edits interact

Concept

BudgetPal separates reusable templates (global definitions) from editable month snapshots (month instances).

Global Definitions

  • Recurring Bills: maintained in Define Global Recurring Bills.
  • Recurring Income: maintained in Define Global Income.
  • Budget Categories: maintained in Define Global Budget Categories.

Month Instances

  • Bills tab edits apply only to selected Year/Month bill occurrences.
  • Income tab edits apply only to selected Year/Month income occurrences.
  • Budget Allocations tab edits apply only to selected Year/Month allocation lines.

Refresh Behavior

ActionEffect
Refresh BillsRebuilds selected month bill instances from global recurring bill definitions.
Refresh IncomeRebuilds selected month income instances from global income definitions.
Refresh Budget CategoriesRebuilds selected month budget allocations from global budget definitions.
Refresh SubscriptionsRe-syncs selected month SubTracker subscription-derived bill rows.

Refresh operations are designed to keep month data aligned with template intent. Run them deliberately for the selected month.

Transaction Import Scope

Spreadsheet imports use an import period key. On re-import of a period, spreadsheet-sourced rows for that period are replaced. This preserves month-level source-of-truth behavior for CFO workflow while retaining historical months in the database.
