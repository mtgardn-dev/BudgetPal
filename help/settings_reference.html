Settings Dialog Tabs

The Settings dialog contains four tabs:

  1. General
  2. Definitions
  3. Accounts
  4. Transfer Rules

Save and Persistence

  • The dialog Save button commits all pending settings values to config.json.
  • Some path changes (especially DB path) apply on next launch.
  • Settings changes are logged to activity log/file logging.
  • Backups are pruned to retain only the most recent configured set.
