Layering

LayerRole
UI (Qt)Widgets, tabs, dialogs, table models, user events.
ServicesBusiness operations: transactions, bills, income, budgeting, subscriptions, reporting.
RepositoriesSQLite CRUD/data queries for each domain area.
PersistenceDB connection, schema, migrations.
ImportersXLSX transaction import and SubTracker view integration.

Core Data Domains

  • Transactions (with import period key, payment type, tax/subscription flags, cleared state).
  • Bills and bill occurrences.
  • Income definitions and income occurrences.
  • Budget category definitions and budget month allocations.
  • Checking month settings and transaction cleared state.
  • Settings, logs, exports, and backups.

Path and Runtime Model

  • BudgetPalPathRegistry is the single source of path resolution.
  • Dev mode writes under project root.
  • Frozen mode writes under OS-appropriate writable app data folders.
  • Bundled resources are resolved from runtime root and bootstrap data.

Release Packaging

  • scripts/app_build.sh builds app bundle and embeds config/images/help/version metadata.
  • scripts/app_make_dmg.sh creates distributable DMG.
  • scripts/release_macos.sh orchestrates build + DMG with a required version string.
