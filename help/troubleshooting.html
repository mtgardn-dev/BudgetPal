App Launches But Window Does Not Appear

  • Check startup error dialogs for DB path/schema failures.
  • Verify BudgetPal DB Path and file permissions in Settings.
  • Review logs for startup exceptions and failed migrations.

XLSX Import Aborts

  • Validate spreadsheet format and required columns.
  • Account values must match defined account aliases (normalization applies but unknown aliases abort import).
  • Check post-import mismatch dialogs/log entries for category type mismatches.

Unexpected Transfer Behavior

  • If transaction matches a transfer rule, rule semantics can override direct spreadsheet intent.
  • Rule-based transfer rows are regenerated on import; manual transfers are managed on Transfers tab.
  • Transfer actions are read-only on Transactions tab by design.
  • Review/adjust rules in Settings > Transfer Rules.

SubTracker Category Issues

  • Run Validate Categories with SubTracker from Settings > General.
  • Fix mismatched category IDs/names between BudgetPal and SubTracker.
  • Re-run Refresh Subscriptions after alignment.

Reconciliation Does Not Balance

  • For checking: verify statement ending/date, cleared flags, and beginning balance.
  • For credit: verify line of credit, statement balance/date, and reported available credit.
  • Use Sort by Transaction ID in Accounts tab if spreadsheet-order comparison is needed.

Reports Issues

  • Some reports require month selection; others are year-only.
  • If files generate but do not open, disable preview and open DOCX files manually.
  • Verify export directory is writable and persisted in Settings/UI preferences.

Backup and Restore Concerns

  • Use Backup Now before refresh/import bulk operations.
  • Exit backups run automatically when backup location is configured.
  • Retention pruning keeps only most recent backups (per current backup policy).
