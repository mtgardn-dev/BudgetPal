First-Run Setup

  1. Launch BudgetPal and click Settings in the header.
  2. Set BudgetPal DB Path and SubTracker DB Path.
  3. Set backup location and base filename.
  4. Confirm logging and window-size settings, then Save.

Monthly Operating Cycle

  1. Select the target Year/Month.
  2. Import transactions from the monthly spreadsheet on Transactions.
  3. Refresh subscriptions on Bills if needed.
  4. Refresh bills, income, and budget categories for the month when ready.
  5. Review and edit month instances.
  6. Use Balance Checking to mark cleared transactions.
  7. Use Reports to preview or export report sets.
  8. Run Send Subscription Payments from Transactions at month end.
