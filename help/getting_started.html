1) First-Run Setup

  1. Open Settings from the header.
  2. On General, set BudgetPal DB Path and SubTracker DB Path.
  3. Set backup location + base name and test with Backup Now.
  4. Configure logging and window size preferences, then Save.
  5. On Definitions, import/export category and global definition files as needed.
  6. On Accounts, define internal/external accounts and line of credit values for credit accounts.
  7. On Transfer Rules, configure rule-based transfer mappings.

2) Monthly Cycle

  1. Select target Year/Month on Dashboard (this syncs tab month selectors).
  2. Import the month spreadsheet from Transactions → Import Transactions.
  3. Review rule-generated transfers on Transfers tab; add/edit manual transfers if needed.
  4. Refresh and review month instances: Bills (Refresh Subscriptions, Refresh Bills), Income (Refresh Income), Budget Allocations (Refresh Budget Categories).
  5. Update month-specific edits (amounts, notes, paid date, category choices, etc.).
  6. Reconcile each account on Accounts tab (checking and credit workflows differ).
  7. Run reports from Reports tab and archive exported DOCX files.
  8. At month end, run Send Subscription Payments from Transactions when needed.

3) Header Buttons

ButtonPurpose

SettingsOpens app configuration, definitions, accounts, and transfer rules.
AboutShows app version/build metadata.
ExitCloses the app and triggers exit-backup logic if configured.

Monthly Safety Checklist

  • Confirm correct Year/Month before any refresh action.
  • Back up before bulk refresh/import actions.
  • Review warnings for category mismatch or account alias validation failures after import.
  • Use Transfers tab for transfer edits; transfer rows are read-only on Transactions.
