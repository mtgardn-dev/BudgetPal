Scope

This is the local, static HTML help system shipped with BudgetPal. It documents current workflow, controls, architecture concepts, integration boundaries, and settings.

File Layout

  • Pages are under help/.
  • Shared styles are under help/css/help.css.
  • Shared logo is help/assets/BudgetPal_Logo_Lg.png.

Packaging

Build scripts include the help directory in the app bundle so Help works in both development and packaged macOS builds.

Maintenance Guidance

  • Update help content in the same change set as UI/behavior changes when possible.
  • Keep UI control names and button labels exact.
  • Verify navigation links after edits.
