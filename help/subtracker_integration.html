Read Contract (Subscriptions)

BudgetPal reads active subscriptions from SubTracker via required view:

v_budgetpal_subscriptions

  • BudgetPal never reads SubTracker base tables directly.
  • BudgetPal validates the view contract/version before and during sync operations.

Refresh Subscriptions Flow

  1. Select Year/Month on Bills tab.
  2. Click Refresh Subscriptions.
  3. BudgetPal reads active subscriptions and upserts month subscription-derived bill rows.
  4. If monthly subscription edits exist, overwrite warning/confirmation is shown first.

Category Mapping Contract

  • Mapping is ID-based between SubTracker and BudgetPal categories.
  • Fallback for unmapped IDs is Uncategorized.
  • BudgetPal logs warnings for mismatches and supports explicit validation from Settings General.
  • Use Validate Categories with SubTracker after category updates.

Write Contract (Subscription Payments)

Send Subscription Payments (Transactions tab) opens a dialog that maps subscription-flagged expense rows to SubTracker subscriptions and posts idempotent payment records.

StepDescription
Select rowsUse per-row checkbox plus Select All/Reset All controls.

Adjust amount (optional)Edit payment amount before posting if needed.
Process PaymentsPosts to SubTracker; reruns are idempotent and update-safe.

Common Integration Failures

  • SubTracker DB path misconfigured in Settings.
  • Missing/invalid v_budgetpal_subscriptions view.
  • Category ID mismatch between systems.
