Tab Order

Dashboard → Transactions → Income → Bills → Budget Allocations → Balance Checking → Savings → Reports.

Dashboard Tab

ControlTypePurpose
Year/MonthCombo boxSelects active month and synchronizes tab month selectors.
Starting BalanceText boxEditable month starting balance used in dashboard calculations.
End Balance / Planned / Actual labelsRead-only labelsSummary of planned/actual income and expenses.
Expenses tableList tableCategory-level planned/actual/diff for expenses.
Income tableList tableCategory-level planned/actual/diff for income.

Transactions Tab

ControlTypePurpose
Year/MonthCombo boxSets month context for transaction list views.
DateText boxTransaction date in YYYY-MM-DD.
AmountText boxCurrency amount for expense or income.
Txn Type (Expense/Income)Radio buttonsControls transaction type sign/behavior.
SubscriptionCheckboxFlags transaction for Sub Payments workflow.
TaxCheckboxGeneral tax flag for the transaction.
DescriptionText boxPrimary narrative description from spreadsheet/manual entry.
CategoryEditable comboSelect/type category with autocomplete.
Account (Cash/Checking/Credit/Savings)Radio buttonsAccount classification.
TypeText boxPayment/channel marker (check #, ACH, card vendor, etc.).
NoteText boxFree-form note for reconciliation context.
AddButtonCreates new transaction and clears form.
SaveButtonUpdates selected transaction.
DeleteButtonDeletes selected transaction.
Import TransactionsButtonImports XLSX monthly transaction sheet.
Send Subscription PaymentsButtonOpens Sub Payments dialog for selected month.
Expenses listList tableExpense rows for selected period.
Income listList tableIncome rows for selected period.

Income Tab

ControlTypePurpose
Year/MonthCombo boxMonth context for income instances.
Description / Start Date / Interval / Amount / Category / Account / NoteText/combosEdit selected monthly income instance.
SaveButtonPersists selected month income edit.
DeleteButtonDeletes selected month income instance row.
Income listList tableMonth income occurrences.
Sort by Category/Description/AccountButtonsSort list by selected key.
Define Global IncomeButtonOpens Income Definitions dialog.
Refresh IncomeButtonRebuilds month instances from global definitions.
Total IncomeSummary labelSum of month income subtotals.
Category Sub-TotalsList tableIncome subtotal by category.

Bills Tab

ControlTypePurpose
Year/MonthCombo boxMonth context for bill instances.
Name / Payment Due / Interval / Amount / Category / NoteText/combosEdit selected monthly bill instance.
SaveButtonPersists selected bill instance edit.
DeleteButtonDeletes selected monthly bill instance.
Bills listList tableMonth bill + subscription occurrences.
Sort by Name/Category/Payment DueButtonsSort list by selected key.
Define Global Recurring BillsButtonOpens recurring bill definitions dialog.
Refresh BillsButtonRebuilds monthly bill instances from global bill definitions.
Refresh SubscriptionsButtonPulls active subscriptions from SubTracker into month instances.
TotalsSummary labelTotal across category subtotals.
Category Sub-TotalsList tableBill subtotal by category.

Budget Allocations Tab

ControlTypePurpose
Year/MonthCombo boxMonth context for budget allocation instances.
Category / Amount / NoteText/combosEdit selected monthly allocation.
SaveButtonPersists allocation row.
DeleteButtonDeletes selected allocation row.
Budget Allocation TotalsList tableAllocation, planned, diff, total planned income, unallocated.
Monthly Budget Category AllocationsList tableCategory-level allocation vs planned bills diff.
Define Global Budget CategoriesButtonOpens budget category definitions dialog.
Refresh Budget CategoriesButtonRebuilds month allocations from global definitions.

Balance Checking Tab

ControlTypePurpose
Year/MonthCombo boxMonth context for checking reconciliation.
Beginning BalanceText boxStarting checkbook value for the month.
SaveButtonSaves beginning balance.
Clear AllButtonMarks all visible rows uncleared.
Reset AllButtonMarks all visible rows cleared.
Checking Transactions and Running BalanceList tableShows checking rows incl. type/date/description/note/amount/running balance/cleared.

Savings Tab

ControlTypePurpose
Add BucketButtonPlanned hook for new savings bucket.
Add MovementButtonPlanned hook for contribution/withdrawal movement.
BucketsList boxBucket list panel.
MovementsList tableLedger for selected bucket.

Reports Tab

ControlTypePurpose
Year / MonthCombo boxesReport period selector (blank month means annual).
Mode (Preview/Export)Radio buttonsPreview opens non-modal viewer; Export writes DOCX.
RunButtonRuns all selected report rows.
Available ReportsMulti-select list tableSelect one or many reports for batch run.
