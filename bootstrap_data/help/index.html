What BudgetPal Does

BudgetPal is a local-first home finance application that tracks monthly cash flow, recurring bills and income, budget allocations, transfers between accounts, and account-level reconciliation.

Primary Capabilities

  • Imports monthly transaction XLSX files from the CFO workflow.
  • Maintains monthly + historical data in a local SQLite database.
  • Manages global definitions and month instances for bills, income, and budget allocations.
  • Supports rule-based and manual transfers with transfer IDs.
  • Provides checking and credit reconciliation views per account tab.
  • Exports DOCX reports and opens them automatically when preview is enabled.

Important Operational Notes

  • Dashboard Year/Month synchronizes the month selectors on other tabs.
  • Refresh Bills, Refresh Income, and Refresh Budget Categories overwrite selected month instances.
  • Transaction import replaces spreadsheet-sourced rows for the imported period key.
  • Rule-based transfers are generated from transfer rules and shown as transfer semantics.
  • Backups run on app exit when backup settings are configured, with retention pruning.
