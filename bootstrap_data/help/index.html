What BudgetPal Does

BudgetPal is a local-first household budgeting application for month-by-month planning, transaction capture, recurring bill/income management, and checking-account balancing. It integrates with SubTracker for subscriptions and subscription-payment posting.

Core Capabilities

  • Imports transaction data from the CFO spreadsheet (XLSX).
  • Maintains historical data across months and years in SQLite.
  • Manages recurring bill and income definitions plus monthly instances.
  • Builds monthly budget allocations by category.
  • Tracks checking-account cleared state and running balance.
  • Generates preview/export reports.

Guiding Model

  • Global definitions describe recurring templates.
  • Month instances are editable snapshots for each Year/Month period.
  • Dashboard summarizes planned vs actual from monthly instances and transactions.
  • SubTracker remains the source of active subscriptions and receives posted payments.

Important Safety Notes

  • Use Year/Month carefully before refreshing monthly instances.
  • Refresh Bills, Refresh Income, and Refresh Budget Categories can overwrite month-level edits.
  • Transaction import replaces spreadsheet-sourced rows for the selected import period key.
  • Use backup settings and Backup Now before major changes.
