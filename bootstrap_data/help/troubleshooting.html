App Starts with No Window

  • Check the activity log file and terminal output for startup warnings.
  • Verify configured DB paths in Settings.
  • If a dialog opened behind windows, use app switcher to reveal it.

SubTracker Warning/Errors

  • Verify SubTracker DB Path.
  • Confirm view v_budgetpal_subscriptions exists.
  • Confirm SubTracker contract version metadata is correct.

Import Results Unexpected

  • Confirm spreadsheet structure matches BudgetPal importer expectations.
  • Ensure you selected correct Year/Month context after import.
  • Remember imports replace spreadsheet-sourced rows for import period key.

Refresh Replaced My Edits

Month instance edits for bills, income, and budget allocations are overwritten when you run corresponding refresh actions. Keep global definitions current first, then refresh month instances.

Backups

  • Set valid backup location in Settings.
  • Use Backup Now before large imports/refreshes.
  • Exit backup failures are logged in activity log.
